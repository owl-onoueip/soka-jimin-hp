# 草加自民・無所属の会議員団 公式サイト

草加市議会「草加自民・無所属の会議員団」の公式ホームページです。  
来年の選挙に向けて、**モバイルファースト**で**支持者獲得**に特化した設計となっています。

---

## 🎯 サイトの目的

### 選挙対策3本柱
1. **認知度向上** - 議員の顔と名前を覚えてもらう
2. **支持者獲得** - 後援会入会・LINE登録の促進
3. **投票促進** - 地域密着アピールで投票行動につなげる

## 📄 ページ一覧

### 1. トップページ (`/`)
- ヒーローセクション（3つの約束）
- 実績数値（要望実現件数、議会質問回数、所属議員数）
- お知らせ一覧
- **議員紹介カルーセル**（自動スクロール）
- 後援会入会バナー
- 関連リンク

### 2. 議員一覧ページ (`/members`)
- 7名の議員カード表示
- 担当地域・キャッチフレーズ・政策タグ
- 会派紹介バナー

### 3. 議員詳細ページ (`/members/[id]`)
- プロフィールヘッダー
- ごあいさつメッセージ
- 重点政策リスト
- 経歴・役職一覧
- 後援会入会CTA

### 4. 後援会ページ (`/support`)
- 後援会の説明（入会費無料・特典）
- 入会申し込みフォーム
- LINE公式アカウント案内

## ✨ 主な機能

### モバイルファースト設計
- 大きなタップターゲット（48px以上）
- スマートフォン優先のレイアウト
- スワイプ操作対応

### 固定CTAボタン
- 画面下部に常時表示
- LINE登録ボタン（緑）
- 後援会入会ボタン（オレンジ）

### 議員カルーセル
- 3秒ごとの自動スクロール
- 無限ループ表示
- タッチ/ホバーで一時停止
- ドットインジケーター

### レスポンシブヘッダー
- モバイル：ハンバーガーメニュー
- PC：横並びナビゲーション

## 🚀 開発・起動方法

### 依存関係のインストール
```bash
npm install
```

### 開発サーバー起動
```bash
npm run dev
```
→ http://localhost:3000 でアクセス

### 本番ビルド
```bash
npm run build
```

### 静的ファイル出力
```bash
npm run export
```
→ `out/` ディレクトリに静的HTMLが生成されます
